# John Stick

Keyboard-only, Three.js **web** brawler — vision in [`docs/GAME_PLAN.md`](docs/GAME_PLAN.md); deferred items in [`docs/TODO.md`](docs/TODO.md). **No backend** in scope.

## Development

```bash
npm install
npm run dev
```

Default dev server port is **5173** (see `vite.config.ts`). Production build: `npm run build`, preview: `npm run preview`.
