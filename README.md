# John Stick

Keyboard-only, Three.js **web** brawler — design spine in [`docs/GAME_PLAN.md`](docs/GAME_PLAN.md). **No backend** in scope.

## Development

```bash
npm install
npm run dev
```

Default dev server port is **5174** (see `vite.config.ts`). Production build: `npm run build`, preview: `npm run preview`.
